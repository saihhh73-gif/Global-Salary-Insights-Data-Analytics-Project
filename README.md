# 🌍 Global Salary Insights Dashboard

## 📌 Project Title

**Global Salary Insights Dashboard – End-to-End Data Analytics Project**



# 📖 Project Overview

The **Global Salary Insights Dashboard** is an end-to-end data analytics project designed to analyze salary trends of data professionals across different countries, experience levels, job categories, company sizes, employment types, and work settings.

The project follows the complete analytics workflow, including data cleaning using Python, business analysis using SQL, and interactive dashboard development in Power BI. The dashboard enables users to explore salary trends, compare compensation across multiple business dimensions, and gain valuable insights through interactive visualizations.



# 🎯 Business Objective

The objective of this project is to:

* Analyze global salary trends in the data industry.
* Compare salaries across different experience levels and job categories.
* Understand the impact of company size and work setting on salaries.
* Identify top-paying countries and highest-paying job roles.
* Build an interactive dashboard for business decision-making.
